Help With Server


Please note that a client can only connect to a server on the same LAN due to firewalls, port connections, and other issues. Thus, a connection over the internet will NOT be implemented.


To initiate a server, being by clicking "host" as you will be hosting a server. Fill in the dedicated fields for "Username" and "Port". The server address field has been pre-filled for you as the server will always be localhosted. Then, click "Launch". The program will check if the port is open and valid. If your input passes this test, you will have successfully created a server on your computer.


Note: The server will continue running as long as the application is not closed out. However, if an error occurs the program will direct you to an error page that will prompt you to either "Exit" or "Restart" the program. Also note, that you will not be able to send any messages until there is at least 1 client connected to your server.


The Kick Feature: As the server's admin/creator, you have the privelege to kick other users. To do so, simply click the user's username in the list. That user's username will appear in the textfield. Then, click the "Kick" button.


Cool Features:

  • Hold Shift+Select Two Users in the List - This will allow you to select a group of all consecutive users between those two selected users (This makes it easier to kick more than one person at once).
  • Hold Ctrl+Select Individual Users - This will allow you to select a group of users that are not consecutive to one another (This makes it easier to kick more than one person at once). Note, using this shortcut on a selected user will deselect them.
  • Enter - When the message field has focus, hitting "Enter" will allow you to quickly send messages without the hassle of clicking the send button.
  • The List of Users - It will auto-update based on who leaves/connects to your server.
  • The Kick TextField - This will display all the users that are selected to be kicked. (It will clear after clicking the "kick" button). In addition, the textfield will update based on selection and/or deselection.

CAUTION: With the knowledge of an IP address and the ID of an open port, an unauthorized person can exploit your connection with the server. SO DO NOT give out your IP Address and Port Number openly. In addition, using the app for wrong-doing/illegal activities will result in your priveleges being revoked. If you attempt to impersonate someone, i.e using their common username the program will simply add a number to the end of your username to differentiate between the two of you.
